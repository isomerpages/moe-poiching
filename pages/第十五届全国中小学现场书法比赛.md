<li>
<p>活动目的：传承华族文化、发扬中华书法艺术。</p>
</li>
<li>
<p>比赛地点：
<br><strong>培青学校 POI CHING SCHOOL<br>21 Tampines Street 71. Singapore 529067<br>【二楼礼堂】</strong>
</p>
</li>
<li>
<p>比赛日期：<strong>2026年5月15日（星期五）</strong>
</p>
</li>
<li>
<p>时间: <strong>下午3时开始（书写时间为一小时）参赛学生必须最迟于下午2时45分报到</strong>。</p>
</li>
<li>
<p>比赛组别:</p>
</li>
</ol>
<p><strong>中学组 </strong>
<br>限在籍的中一至中四年级的学生
<br><strong>小学组</strong>
<br>限在籍的一至六年级的学生</p>
<ol start="6" data-tight="true" class="tight">
<li>
<p>报名详情：</p>
</li>
</ol>
<p>• 比赛将进行现场书写，并无初赛阶段。
<br>• 本次比赛只能通过学校报名。
<br>• 每所学校的参赛学生最多12人
<br>• 参赛学校的负责老师必须通过formSG电子表格报名并于4月20日之前呈交。
<br>• 一旦成功完成报名，电子表格将出现“Thank you for filling out the form”以及表格的RESPONSE ID，系统也会通过电邮自动转发报名的收据给予报名的老师。主办单位将不会另行电邮通知各校报名结果。</p>
<ol start="7" data-tight="true" class="tight">
<li>
<p>报名链接：</p>
</li>
</ol>
<p><a href="https://go.gov.sg/poiching-15-may-2026" rel="noopener noreferrer nofollow" target="_blank">https://go.gov.sg/poiching-15-may-2026</a>
</p>
<ol start="8" data-tight="true" class="tight">
<li>
<p>比赛详情：书写工具</p>
</li>
</ol>
<p>• 参赛者须自备毛笔、墨、墨碟、练习纸张、垫布等。
<br>• 主办单位会分发两张宣纸 (未折) 给每位参赛同学。
<br>• 同学须在比赛前掌握折纸方法
<br>• 宣纸尺寸为34cmX69cm。
<br>• 参赛学生的参赛作品必须用所分发的宣纸书写。
<br>• 字帖（包括手机内存图片）不得带入赛场。
<br>• 每位参赛同学只能呈交一件作品。</p>
<ol start="9" data-tight="true" class="tight">
<li>
<p>赛题：</p>
</li>
</ol>
<p>• 比赛题目会在当天分发。
<br>• 小学组为5字对联；中学组为7字对联。</p>
<ol start="10" data-tight="true" class="tight">
<li>
<p>书体：参赛学生可用任何书体书写。</p>
</li>
<li>
<p>奖励：</p>
</li>
</ol>

<br>第一名：奖状及书券120元
<br>第二名：奖状及书券 100元
<br>第三名：奖状及书券 80元
<br>优胜奖(15名)：奖状及书券30元
<br>安慰奖(20名)：奖状及书券20元</p>
<p></p>
<p><strong>小学组</strong>
<br>第一名：奖状及书券 100元
<br>第二名：奖状及书券 80元
<br>第三名：奖状及书券 60元
<br>优胜奖(15名)：奖状及书券30元
<br>安慰奖(20名)：奖状及书券20元</p>
<ol start="12" data-tight="true" class="tight">
<li>
<p>评审：由校外书法界知名人士担任，作品一经评定，即为最后决定。</p>
</li>
<li>
<p>成绩揭晓及领奖方式：</p>
</li>
</ol>
<p>• 成绩将于<strong>2026年 5月25日（星期一）</strong>通过电邮方式通知各所参赛学府。
<br>•比赛结果也会上载到培青学校的官方网站。
<br>• 得奖的同学（或负责老师代领）必须在2026年8月7日（星期五）之前，前往培青学校办公室领取奖状以及书券，否则当自动放弃奖项。</p>
<ol start="14" data-tight="true" class="tight">
<li>
<p>其他事项：</p>
</li>
</ol>
<p>•参赛学生必须穿校服参赛并保持会场的清洁。
<br>•比赛正式开始后，带队老师以及家长须移玉步至一楼食堂等候。
<br>•由于校园内停车位有限，请开车到会场的带队老师和家长把车辆停泊在附近的多层停车场。</p>
